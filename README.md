# Prime Green at Arlington

Starter site for a community sustainability hub covering Arlington County, VA: solar and renewables, recycling, climate goals, education, and independent consulting.

## Files

- `index.html` - homepage: hero, solar, interactive county map, recycling guide, schools, zigzag 2050 timeline, fork-in-the-road section, consulting teaser; a dotted trail walks down the page like a map
- `consulting.html` - solar planning and clean energy education consulting
- `prime.css` - shared styles
- `.nojekyll` - tells GitHub Pages to serve files as-is

## Publish on GitHub Pages

1. Create a new GitHub repo and push this folder's contents to the root of the default branch.
2. In the repo: Settings, then Pages, then set Source to "Deploy from a branch" and pick the default branch, root folder.
3. The site appears at `https://<username>.github.io/<repo>/` in a minute or two.
